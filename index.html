JS Шпаргалка в формате TV‑каналов

1. Где писать код?

Консоль браузера и тег <script> — стартовая точка.

<!-- В консоли браузера (F12) или в теге <script> -->
<script>
  console.log('Привет, мир!');
</script>

Открой DevTools (F12) → вкладка Console и пробуй команды прямо там.

2. Переменные (контейнеры для данных)

Три способа объявить переменную, но в 2026 году хватит let и const.

// Три способа создать переменную
let name = 'Анна';        // Можно менять позже
const PI = 3.14;          // Нельзя изменить (константа)
var old = 'Так лучше не делать'; // Устаревший способ

По умолчанию используй const, а let — когда значение реально меняется.

3. Типы данных (самые частые)

JS — динамически типизированный, но базовые типы знать всё равно полезно.

let number = 42;          // Число
let string = "Текст";     // Строка (в кавычках)
let isTrue = true;        // Булево (true/false)
let nothing = null;       // Пусто (намеренно)
let notDefined;           // undefined (не присвоено значение)

Проверить тип поможет typeof value в консоли.

4. Операторы

Математика и логика — два главных инструмента в условиях и вычислениях.

// Математика
5 + 2  // 7
5 - 2  // 3
5 * 2  // 10
5 / 2  // 2.5

// Логика (сравнение)
5 === 5   // true (строго равно, проверяет и значение и тип)
5 !== 5   // false (не равно)
5 > 3     // true
5 <= 5    // true

// Для новичков: всегда используй === и !==, а не == и !=

Запомни правило: === и !== почти всегда лучше, чем == и !=.

5. Условные конструкции (если — то)

Ветвление позволяет выполнять разный код при разных условиях.

let age = 20;

if (age >= 18) {
  console.log('Проходи');
} else if (age >= 16) {
  console.log('Почти');
} else {
  console.log('Рано');
}

Внутри блока можно вызывать функции, менять переменные и делать что угодно.

6. Функции (кирпичики программы)

Функция — именованный фрагмент кода, который можно переиспользовать.

// 1. Объявление функции
function greet(name) {
  return 'Привет, ' + name;
}

// 2. Вызов функции
let message = greet('Петя'); // message = 'Привет, Петя'

// Стрелочная функция (современный стиль)
const sum = (a, b) => a + b;
let result = sum(5, 3); // 8

Если функция делает слишком много, разбей её на несколько маленьких.

7. Массивы (списки)

Массив хранит упорядоченную коллекцию значений.

let fruits = ['Яблоко', 'Банан', 'Апельсин'];

console.log(fruits[0]);      // 'Яблоко' (нумерация с нуля)
console.log(fruits.length);  // 3 (длина массива)

fruits.push('Груша');        // Добавить в конец

Для перебора массивов часто используют for...of, forEach, map, filter.

8. Циклы (повтор действий)

Цикл позволяет пройтись по массиву или повторить одно действие много раз.

// Цикл для перебора массива
let colors = ['red', 'green', 'blue'];

for (let i = 0; i < colors.length; i++) {
  console.log(colors[i]);
}

// Современный способ (для массивов)
for (let color of colors) {
  console.log(color);
}

Следи, чтобы условие цикла когда-нибудь стало ложным — иначе получится бесконечный цикл.

9. Объекты (сущности со свойствами)

Объект описывает сущность: у неё есть свойства и значения.

let user = {
  name: 'Иван',
  age: 25,
  isAdmin: false
};

console.log(user.name);      // 'Иван' (через точку)
console.log(user['age']);    // 25 (другой способ)
user.city = 'Москва';        // Добавить новое поле

Почти всё в JS так или иначе работает с объектами — это ключевая структура данных.

10. Работа с DOM и полезные мелочи

DOM — это JS‑представление HTML‑страницы, с которым можно работать из кода.

// Найти элемент на странице
let title = document.getElementById('header');
let buttons = document.querySelectorAll('.btn'); // CSS селекторы

// Изменить элемент
title.textContent = 'Новый текст';  // Поменять текст
title.style.color = 'red';          // Поменять стиль

// Создать новый элемент
let div = document.createElement('div');
div.textContent = 'Я новый блок';
document.body.appendChild(div);     // Вставить в конец страницы

// Реакция на клик
button.addEventListener('click', () => {
  alert('Клик!');
});

// Шаблонные строки (вставляем переменные в текст)
let name = 'Мир';
console.log(`Привет, ${name}!`); // Привет, Мир! (обратные кавычки)

// Тернарный оператор (короткое if)
let access = (age >= 18) ? 'Да' : 'Нет';

// Очистка консоли
console.clear();

Главный совет: открывай консоль (F12) и пробуй каждую строчку руками. Ошибки — это нормально.
